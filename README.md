# ContainerPacker3D

A 3D container packing visualization tool built with Three.js. Supports interactive rotation, zoom, box placement, and exporting packed boxes as JSON.

基于 Three.js 的 3D 装柜可视化工具，支持交互旋转、缩放、箱子摆放和导出 JSON。
